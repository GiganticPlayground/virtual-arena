How to read the mask types

Categorical (Silhouette / Multiclass) emits an integer class per pixel; the compositor thresholds it. Hard edges.

Continuous alpha (RVM / U²-Net-p / MODNet / Silueta) emits a float matte; the compositor averages it through the feather kernel for soft edges.

How to pick one

Default demo: Silhouette. Best interactive soft edge: U²-Net-p. Best sharpness at interactive FPS: MODNet. Best temporal stability on video: RVM (once multithreaded WASM is available). Silueta: screenshot-only reference.
